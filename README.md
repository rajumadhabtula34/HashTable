# UC3 - Remove a Word from Hash Table

## Problem Statement

Implement the remove operation in the Hash Table to delete the word **"avoidable"** from the paragraph.

## Objective

Enhance the custom Hash Table by supporting deletion of a key-value pair from the appropriate bucket using a LinkedList.

## Features

* Delete a word from the Hash Table.
* Locate the correct bucket using `hashCode()`.
* Remove the corresponding node from the LinkedList.
* Display the updated Hash Table after deletion.

## Technologies Used

* Java
* Java Generics
* Hash Table
* Linked List
* Object-Oriented Programming
